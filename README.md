# Prediksi-Interval-Harga-Saham-LQ45-Dengan-LSTM-Dan-Quantile-Regression
Penelitian ilmiah yang membangun model prediksi **interval harga saham** pada indeks LQ45 Bursa Efek Indonesia. Model ini tidak hanya memprediksi **arah pergerakan harga** (naik/turun), tetapi juga menghasilkan **batas bawah dan batas atas**
